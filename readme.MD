---

## Navigate into the project

```bash

```

---

## Install dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a `.env` file in the root directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_access_token_secret

JWT_REFRESH_SECRET=your_refresh_token_secret

EMAIL_USER=yourgmail@gmail.com

EMAIL_PASS=your_gmail_app_password
```

---

## Run the development server

```bash
npm run dev
```

Server runs on:

```
http://localhost:5000
```

# рџ”‘ API Endpoints

## Register User

**POST**

```
/api/auth/register
```

Body

```json
{
  "firstName": "John",
  "lastName": "Doe",
  "email": "john@example.com",
  "password": "password123"
}
```

---

## Verify Email

**GET**

```
/api/auth/verify/:token
```

Users verify their account by clicking the verification link sent to their email.

---

## Login

**POST**

```
/api/auth/login
```

Body

```json
{
  "email": "john@example.com",
  "password": "password123"
}
```

Response

```json
{
  "accessToken": "...",
  "refreshToken": "..."
}
```

---

## Protected Profile

**GET**

```
/api/auth/profile
```

Headers

```
Authorization: Bearer ACCESS_TOKEN
```

---

## Forgot Password

**POST**

```
/api/auth/forgot-password
```

Body

```json
{
  "email": "john@example.com"
}
```

A password reset link is sent to the registered email.

---

## Reset Password

**POST**

```
/api/auth/reset-password/:token
```

Body

```json
{
  "password": "newpassword123"
}
```

---

## Refresh Access Token

**POST**

```
/api/auth/refresh-token
```

Body

```json
{
  "refreshToken": "YOUR_REFRESH_TOKEN"
}
```

Returns a brand-new Access Token.

---

## Logout

**POST**

```
/api/auth/logout
```

Body

```json
{
  "refreshToken": "YOUR_REFRESH_TOKEN"
}
```

Invalidates the current session by removing the Refresh Token from the database.
